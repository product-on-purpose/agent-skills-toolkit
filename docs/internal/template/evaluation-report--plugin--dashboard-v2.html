02

Executive summary

For a reader who is not in the code: what this library is, how it scored, and the one move that moves it forward.

How to read this report

The colored matrix above is the whole verdict in one glance: each of the 30 checks is a chip, color-coded pass / fail / warn / not-applicable, grouped by tier. The tier a library earns is decided by a deterministic gate (a real program with a real exit code), not by opinion. Everything below the matrix expands that picture: a plain-language summary, the full component inventory, the methodology, a per-check evidence ledger, the exact work that blocks the next tier, and copy-paste prompts that fix each gap.

acme-content-skills is a content-marketing toolkit of 12 skills, 2 review subagents, and 1 slash command, built to run on both Claude Code and Codex. It declared the convergent (Silver) tier, and it earns exactly that. Of the 30 checks in the Standard, 27 pass. The deterministic gate exits 0, which is the machine-readable way of saying the library meets every requirement of the tier it claimed. There are no surprises hiding behind the badge: Bronze (the universal floor) and Silver (clean convergence across both declared agents) are fully satisfied.

The two advisory warnings are quality nudges, not failures, and they do not affect the tier. One skill, acme-content-calendar, has a description that scores 0.62 against a 0.70 bar because it names neither a concrete action nor a use-when trigger, which makes it harder for an agent to route to it. A second skill, acme-write-blog-post, runs about 240 instructions, over the roughly 200 guidance, so its later steps (the SEO pass) risk being dropped when context gets tight. Both are ten-minute fixes and both have a ready prompt in section 07.

The interesting story is the climb to Gold. Seven of the ten Advanced checks already pass, including the entire documentation taxonomy that landed in this Standard version (folder READMEs, source docblocks, a full Diataxis docs tree, and docs frontmatter). What stands between this library and Gold is the self-proving discipline: there is no CI workflow that runs the gate (G2), two of the three chain edges have no regression eval (G3), and there is no curated, user-facing RELEASE-NOTES.md distinct from the changelog (G5). None of these are about the skills being weak; they are about the library being able to prove and maintain itself over time.

Recommended next move: treat Gold as a single short session. Add the CI workflow that shells out to the gate (G2, about 20 minutes), add the two missing chain evals (G3, about 30 minutes), and write the release notes (G5, about 15 minutes). That is roughly one focused hour, after which a re-run of the gate should report Gold. Fold in the two warnings while you are there and the library is both higher-tier and cleaner. Concrete, copy-paste prompts for all five items are in the improvement path.

04

Methodology and scope

Three layers, kept strictly separate so the verdict stays honest. Only one of them decides the tier.

Layer 1 - Conformance

Decides the tier. The portable Node gate runs all 30 checks with real exit codes, no model in the loop. This is the only layer that decides pass or fail. Findings are exact and reproducible: re-run node scripts/check.mjs . to confirm.

Layer 2 - Behavioral

Advisory only. Each skill is run against its triggering eval set; results are surfaced as evidence and never decide a gate result. Reflects judgment on sampled cases, not a guarantee.

Layer 3 - Review

Advisory only. A qualitative pass over scoping, descriptions, and chain design. Informs the insights in section 08; it does not move the badge.

Status legend

Confidence and limitations

Conformance findings are exact and reproducible: the same gate on the same commit yields the same result. Behavioral and review findings reflect judgment on sampled cases and may shift with different inputs. Three checks pass vacuously because the library contains nothing for them to test: G1 (no hooks), G6 (no deprecations), and U11 (no MCP servers). A vacuous pass is a genuine pass under the Standard, not a gap.

What each tier guarantees

Bronze - Universal

The library is portable and self-describing: it installs and reads correctly on any agentskills.io-compatible agent, with valid anatomy, frontmatter, and naming.

Silver - Convergent

On top of Bronze, the library converges cleanly across every agent it declares: prefixes, native manifests, chain contracts, and the components index all agree across targets.

Gold - Advanced

On top of Silver, the library is self-proving: it builds, checks, documents, and releases itself - CI runs the gate, chains are regression-covered, docs are complete, and releases are curated.

08

Insights

Qualitative observations beyond pass and fail, from the advisory review layer.

01

The library is one short session from Gold

All three blockers are infrastructure, not craft: CI, chain evals, and release notes. The skills themselves are already documented to the new taxonomy (G7 to G10 all pass). This is a mature library missing only the self-proving wrapper, which is unusually close.

02

Documentation is a genuine strength

Folder READMEs, source docblocks, docs frontmatter, and a full Diataxis tree all pass. Many libraries at this size skip the explanation quadrant; this one carries it. The docs investment already made will make the Gold climb feel disproportionately cheap.

03

The two warnings share a root cause: scoping discipline

A vague description (U5) and an over-budget body (U7) both point to skills that try to do or say a little too much in one place. Tightening the calendar description and pushing the blog-post SEO detail into a reference are small edits that also improve day-to-day routing and reliability.

04

Chain design is sound; only the safety net is missing

The three edges are sensible (writing and repurposing both delegate to the style critic; writing also fact-checks). The contract is clean (S4 passes). The gap is purely that two edges lack a regression eval, so a future change to the style critic could break two callers unnoticed.

05

Cross-agent posture is honest

Skills target both agents; subagents and the command are Claude-only and declared as such. Nothing pretends to be more portable than it is, which is exactly what Silver should look like. There is no convergence debt to pay down before Gold.
