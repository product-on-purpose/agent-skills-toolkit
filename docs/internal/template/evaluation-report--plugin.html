01Executive summary

acme-content-skills is a healthy, genuinely cross-agent plugin that has cleared the multi-agent bar and now sits one well-scoped push away from the self-proving Gold tier.

The plugin was graded against all 26 deterministic checks of the Standard, plus a behavioral triggering pass and a qualitative review. It earns Silver (Convergent): its 12 skills, 2 subagents, and 1 command are portable, carry a consistent acme- prefix, declare both claude and codex targets, and emit clean per-agent manifests. Two Bronze warnings (a weak description and an over-budget skill) do not gate the build but are worth closing.

Gold is blocked by three checks, all structural rather than deep: there is no self-hosting CI that runs the gate (G2), two chains lack a regression eval (G3), and there is no curated RELEASE-NOTES.md (G5). Each is closeable in well under a day; the Improvement path below gives a copy-paste prompt for every one.

Recommended next move: run the three Gold-blocker prompts in order (CI, evals, release notes), then re-evaluate. That promotes the plugin to a self-proving Gold library without touching a single skill's behavior.

03Methodology & scope

Three layers, kept strictly separate so the verdict stays trustworthy:

  • Deterministic conformance (decides the tier). The portable Node gate ran all 26 checks (U1-U11, S1-S8, G1-G7) with real exit codes, no model in the loop. This is the only layer that decides a pass or fail.
  • Behavioral (advisory). Each skill was run against its triggering eval set to confirm it fires when it should and stays silent when it should not. Surfaced as evidence, never as a gate result.
  • Review (advisory). A qualitative pass over scoping, descriptions, and chain design.
pass fail (gates at the declared tier) warning (never gates) not applicable / vacuous

Confidence and limits. Conformance findings are exact and reproducible (re-run the gate to confirm). Behavioral and review findings reflect judgment on the sampled cases. G1 (hooks) and G6 (deprecation) pass vacuously here because the plugin ships no hooks and deprecates nothing yet.
