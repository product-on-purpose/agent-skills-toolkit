# Decisions (ADRs)

Architecture Decision Records in MADR format, numbered and immutable once
accepted (Standard sec 10.4). Each ADR carries a `## TL;DR` (decision, why,
status), which the `docs-presence` check (G10) enforces.

The consolidated early decision set D1-D19 lives in [`../DESIGN.md`](../DESIGN.md);
individually numbered records graduated here start at `0020`.
