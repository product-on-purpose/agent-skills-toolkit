# templates

The seed templates the askit-* skills scaffold from: a Bronze-ready plugin seed plus a starter per component type.

## Inventory

- `SKILL.md` - the skill definition (frontmatter plus the procedure).
- `adr.md` - the adr component template.
- `agent.md` - the agent component template.
- `chain-permitted.yaml` - the chain-permitted.yaml config.
- `command.md` - the command component template.
- `eval-set.json` - the eval-set.json config.
- `hooks.json` - the hook registration (event, matcher, action).
- `mcp.json` - the mcp.json config.
- `onboarding-questionnaire.template.md` - the onboarding-questionnaire.template component template.
- `output-style.md` - the output-style component template.
- `seed-plugin/` - the seed-plugin entry.
- `workflow.md` - the workflow component template.
