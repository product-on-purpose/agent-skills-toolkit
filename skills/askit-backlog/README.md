# skills/askit-backlog

The askit-backlog skill. Creates, triages, and prunes a plugin's two backlogs (new-component proposals and enhancements) to the Advanced Skill Library Standard. Use when capturing a new-component proposal through the why-gate, prioritizing backlog items, or removing stale or completed entries.

## Inventory

- `SKILL.md` - the skill definition (frontmatter plus the procedure).
- `references/` - supporting reference docs loaded on demand.
