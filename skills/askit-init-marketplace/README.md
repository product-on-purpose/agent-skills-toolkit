# skills/askit-init-marketplace

The askit-init-marketplace skill. Creates and validates a marketplace index that catalogs plugins for Claude Code and Codex, checking each entry, its plugin reference, and version consistency. Use when standing up a marketplace, adding a plugin to a marketplace, or validating marketplace entries.

## Inventory

- `SKILL.md` - the skill definition (frontmatter plus the procedure).
- `references/` - supporting reference docs loaded on demand.
