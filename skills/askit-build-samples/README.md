# skills/askit-build-samples

The askit-build-samples skill. Creates and validates a skill's sample sets and eval sets (golden examples, anti-examples, and triggering cases) and detects drift against current behavior. Use when generating samples for a skill, building an eval set, or checking samples for drift.

## Inventory

- `SKILL.md` - the skill definition (frontmatter plus the procedure).
- `references/` - supporting reference docs loaded on demand.
