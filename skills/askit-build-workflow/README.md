# skills/askit-build-workflow

The askit-build-workflow skill. Creates and improves workflows (an ordered multi-skill arc) for a plugin to the Advanced Skill Library Standard. Use when a recurring sequence of skills is worth formalizing as a workflow, when you need to author a _workflows file with steps and exit criteria, or to resolve S5 workflow findings.

## Inventory

- `SKILL.md` - the skill definition (frontmatter plus the procedure).
- `references/` - supporting reference docs loaded on demand.
