# skills/askit-migrate

The askit-migrate skill. Assesses an existing skills repository against the Advanced Skill Library Standard, produces a staged bring-to-conformance plan, and writes the minimal canonical manifest so the repo becomes gradeable. Use when adopting a foreign or ad-hoc skills repo, migrating a Claude-only plugin toward cross-agent conformance, or planning a Bronze-to-Silver upgrade path.

## Inventory

- `SKILL.md` - the skill definition (frontmatter plus the procedure).
- `references/` - supporting reference docs loaded on demand.
